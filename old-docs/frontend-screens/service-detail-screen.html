Service Detail Screen

Description

The Service Detail Screen provides comprehensive information about a specific service offering. It showcases the professional's work through images, detailed descriptions, pricing, availability, and customer reviews. This screen is crucial for converting browsing users into bookings by providing all necessary information to make an informed decision.

Primary Purpose: Present complete service information to help users evaluate and book services, while building trust through reviews, professional credentials, and detailed service specifications.

UI Elements

  • Header Section:
    • Back navigation button
    • Share button
    • Favorite/Save button
  • Image Gallery:
    • Main image carousel
    • Thumbnail navigation
    • Full-screen view option
    • Image count indicator
  • Service Information:
    • Service title
    • Category and subcategory tags
    • Price range with breakdown
    • Duration estimate
    • Service area/coverage
  • Professional Card:
    • Professional photo and name
    • Verification badge
    • Overall rating and review count
    • Response time
    • Years of experience
    • "View Profile" button
    • "Contact" button
  • Description Section:
    • Detailed service description
    • What's included list
    • Requirements from customer
    • Cancellation policy
  • Availability Calendar:
    • Monthly view with available dates
    • Time slot selector
    • Timezone indicator
  • Reviews Section:
    • Rating breakdown (5-star distribution)
    • Sort options (Most recent, Most helpful)
    • Individual review cards with photos
    • "Load more" button
  • Sticky Bottom Bar:
    • Price summary
    • "Book Now" CTA button
    • "Request Quote" alternative
  • Related Services: Horizontal scroll of similar services

API Calls

  • services.getById(serviceId) - Fetch complete service details
  • professionals.getProfile(professionalId) - Get professional information
  • reviews.getByService(serviceId, pagination) - Load service reviews
  • availability.getCalendar(serviceId, month) - Get availability calendar
  • services.getRelated(serviceId) - Fetch similar services
  • favorites.toggle(serviceId) - Add/remove from favorites
  • analytics.trackView(serviceId) - Track service view for recommendations

State Management

  • Service State: Complete service details, loading status
  • Gallery State: Current image index, fullscreen mode
  • Calendar State: Selected date, available slots, timezone
  • Review State: Reviews array, current page, sort order
  • UI State: Favorite status, share modal, login modal
  • Booking State: Selected date/time, price calculation

Dynamic Content

  • Price Calculation: Updates based on selected options/duration
  • Availability Updates: Real-time slot availability checking
  • Review Updates: New reviews appear without page refresh
  • View Count: Increment and display popularity metrics
