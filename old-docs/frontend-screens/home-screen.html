Home Screen

Description

The Home Screen is the main landing page of the services marketplace. It serves as the entry point for both customers looking for services and professionals showcasing their offerings. The screen provides an intuitive interface for quick service discovery through featured services, search functionality, and category browsing.

Primary Purpose: Enable users to quickly find services they need through multiple discovery methods while showcasing popular and featured services to drive engagement.

UI Elements

  • Navigation Bar: Logo, Search bar, Login/Register buttons (or User profile if logged in)
  • Hero Section: Welcome message, main search bar with autocomplete, quick category buttons
  • Featured Services Carousel: Horizontally scrollable cards showing top-rated services
  • Service Categories Grid: Icon-based grid showing all available service categories
  • Popular Services Section: List of most booked services in the area
  • How It Works Section: 3-step process explanation for new users
  • Footer: Links to About, Help, Terms, Contact

API Calls

  • services.getFeatured() - Fetches featured services for carousel
  • categories.getAll() - Retrieves all service categories
  • services.getPopular() - Gets popular services based on bookings
  • search.getSuggestions(query) - Provides autocomplete suggestions

State Management

  • Loading States: Skeleton loaders for featured services and categories while data loads
  • Error States: Fallback UI if API calls fail, with retry options
  • Empty States: Helpful messages if no services are available in user's location
  • Authentication State: Different UI elements based on logged in/out status
