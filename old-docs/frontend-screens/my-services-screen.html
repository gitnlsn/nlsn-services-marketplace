My Services Screen

Description

The My Services Screen provides professionals with a comprehensive overview of all their service listings, enabling efficient management of their marketplace portfolio. This interface serves as the central hub for service administration, performance monitoring, and optimization activities.

Primary Purpose: Enable professionals to effectively manage their service portfolio, monitor performance metrics, and take actions to optimize their marketplace presence and earnings.

UI Elements

  • Header Actions:
    • "Create New Service" primary button
    • Bulk actions dropdown
    • View toggle (grid/list)
    • Sort options (date, performance, status)
  • Service Status Filters:
    • All Services
    • Active
    • Inactive/Paused
    • Draft
    • Under Review
    • Archived
  • Service Cards/List Items:
    • Service thumbnail image
    • Service title and category
    • Status indicator and badge
    • Performance metrics (views, bookings, rating)
    • Last updated timestamp
    • Earnings summary
    • Quick action buttons
  • Quick Actions:
    • Edit service
    • View analytics
    • Duplicate service
    • Pause/unpause
    • Archive service
    • Share service link
  • Performance Summary:
    • Total services count
    • Total monthly earnings
    • Average service rating
    • Conversion rate
  • Search and Filter:
    • Service name search
    • Category filter
    • Date range picker
    • Performance filters

API Calls

  • services.getByProfessional(professionalId, filters) - Fetch professional's services
  • services.updateStatus(serviceId, status) - Change service status
  • services.duplicate(serviceId) - Create copy of existing service
  • services.archive(serviceId) - Archive service
  • analytics.getServiceSummary(professionalId) - Get performance overview
  • analytics.getServiceMetrics(serviceId) - Individual service performance
  • services.bulkUpdate(serviceIds, updates) - Bulk operations
  • bookings.getCountByService(serviceIds) - Get booking counts

Service Management Features

  • Status Management:
    • Active: Visible to customers, accepting bookings
    • Paused: Hidden from search, existing bookings maintained
    • Draft: Unpublished, still being created
    • Under Review: Pending platform approval
    • Archived: No longer available, historical data preserved
  • Performance Tracking:
    • View counts and click-through rates
    • Booking conversion metrics
    • Customer rating trends
    • Revenue per service

Optimization Tools

  • Performance Insights: AI-powered suggestions for improving service visibility and bookings
  • Competitive Analysis: Compare service performance against category averages
  • A/B Testing: Test different versions of service listings
  • SEO Recommendations: Suggestions for improving search ranking

Bulk Operations

  • Status Changes: Pause/unpause multiple services
  • Pricing Updates: Apply percentage increases across services
  • Category Migration: Move services to different categories
  • Template Application: Apply consistent formatting/policies
