Create/Edit Service Listing

graph TD
    A[Frontend Request: Create/Edit Service]:::frontend --> B{Authentication & Authorization Check}:::auth
    B -- Success --> C[Validate Service Data]:::validation
    B -- Failure --> D[Return 401/403 Error]:::error
    C -- Valid --> E{Check if Existing Service}
    C -- Invalid --> F[Return 400 Error: Validation Failed]:::error
    E -- Yes --> G[Call Prisma: Update Service Record]:::prisma
    E -- No --> H[Call Prisma: Create Service Record]:::prisma
    G -- Success --> I[Return 200 OK: Service Updated]:::success
    G -- Failure --> J[Return 500 Error: Database Error]:::error
    H -- Success --> K[Return 201 Created: Service Created]:::success
    H -- Failure --> J
