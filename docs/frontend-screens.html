Frontend Screens and Navigation Flow

This document outlines the key frontend screens of the application and illustrates the navigation flow between them, distinguishing between customer and professional user experiences.

1. Customer Screens

  • Home Screen: Landing page with featured services, search bar, and categories.
  • Search Results Screen: Displays services based on search queries or category selection.
  • Service Detail Screen: Detailed view of a specific service, including description, pricing, photos, and professional's profile link.
  • Professional Profile Screen: Displays the professional's public profile, including all their services and reviews.
  • Booking Confirmation Screen: Summary of a booked service before payment.
  • Payment Screen: Interface for completing the payment for a booked service.
  • My Bookings Screen (Customer): Lists all services booked by the customer.
  • Reviews Screen: Allows customers to leave reviews for completed services.

2. Professional Screens

  • Professional Dashboard: Overview for professionals, showing active services, pending bookings, and earnings.
  • My Services Screen: Lists all services created by the professional, with options to edit or deactivate.
  • Create Service Screen: Form for professionals to create new service listings.
  • Edit Service Screen: Form for professionals to modify existing service listings.
  • My Bookings Screen (Professional): Lists all bookings received by the professional.
  • Earnings & Withdrawal Screen: Displays earnings, available balance, and allows withdrawal requests.
  • Professional Profile Screen (Edit): Allows professionals to edit their public profile information.
