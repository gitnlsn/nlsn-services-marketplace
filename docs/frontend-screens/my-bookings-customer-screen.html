My Bookings Screen (Customer)

Description

The My Bookings Screen for customers provides a comprehensive view of all their service bookings across different stages of completion. This screen serves as the central hub for customers to track their appointments, manage upcoming services, view booking history, and take actions like rescheduling or leaving reviews.

Primary Purpose: Enable customers to efficiently manage their service bookings, track appointment status, and access booking-related actions all in one place.

UI Elements

  • Filter Tabs:
    • All Bookings
    • Upcoming (Confirmed/Pending)
    • Completed
    • Cancelled
  • Booking Cards:
    • Service thumbnail and title
    • Professional name and photo
    • Booking status badge
    • Date and time
    • Service location/address
    • Total price paid
    • Booking reference number
  • Status-Specific Actions:
    • Pending: "Cancel Booking", "Edit Details"
    • Confirmed: "Reschedule", "Cancel", "Contact Professional"
    • Completed: "Leave Review", "Book Again", "Download Receipt"
    • Cancelled: "Book Again", "View Refund Status"
  • Search and Sort:
    • Search by service or professional name
    • Sort by date, status, or price
    • Filter by date range
  • Empty States: "No bookings yet" with CTA to browse services
  • Pagination: For customers with many bookings

API Calls

  • bookings.getByCustomer(customerId, filters) - Fetch customer's bookings
  • bookings.cancel(bookingId, reason) - Cancel a booking
  • bookings.reschedule(bookingId, newDateTime) - Request reschedule
  • reviews.create(bookingId, rating, comment) - Submit service review
  • receipts.download(bookingId) - Generate PDF receipt
  • refunds.getStatus(bookingId) - Check refund processing status
  • chat.initiate(professionalId) - Start conversation with professional

Booking Status Management

  • Pending: Awaiting professional confirmation
  • Confirmed: Professional accepted, service scheduled
  • In Progress: Service is currently being performed
  • Completed: Service finished, payment processed
  • Cancelled: Booking cancelled by customer or professional
  • Refunded: Payment refunded due to cancellation

Review System Integration

  • Review Prompts: Automatic prompts for completed services
  • Review Requirements: 1-5 star rating + optional written review
  • Photo Upload: Allow customers to upload photos with reviews
  • Review Editing: Allow edits within 24 hours of submission

Notification Integration

  • Status Updates: Real-time notifications for booking changes
  • Reminders: Upcoming appointment notifications
  • Review Requests: Gentle prompts to leave reviews
  • Professional Messages: Communications from service providers
