Professional Dashboard

Description

The Professional Dashboard serves as the central command center for service providers, offering a comprehensive overview of their business performance, active services, booking requests, and earnings. This dashboard enables professionals to efficiently manage their marketplace presence, track key metrics, and make informed decisions about their service offerings.

Primary Purpose: Provide professionals with actionable insights and streamlined management tools to optimize their service delivery, maximize earnings, and maintain high customer satisfaction.

UI Elements

  • Header Summary Cards:
    • Total Earnings (this month)
    • Pending Bookings count
    • Active Services count
    • Customer Rating average
    • Response Rate percentage
  • Quick Action Buttons:
    • Create New Service
    • View All Bookings
    • Request Withdrawal
    • Update Profile
    • View Analytics
  • Pending Bookings Section:
    • List of recent booking requests
    • Quick accept/decline actions
    • Time remaining to respond
    • "View All" link to full bookings page
  • Active Services Widget:
    • Service performance metrics
    • View/booking counts
    • Edit/deactivate options
    • Service status indicators
  • Earnings Overview:
    • Monthly earnings chart
    • Available balance
    • Pending payments
    • Withdrawal history
  • Performance Metrics:
    • Customer reviews summary
    • Booking completion rate
    • Average response time
    • Service popularity trends
  • Notifications Panel: Platform updates, policy changes, tips for improvement

API Calls

  • dashboard.getSummary(professionalId) - Fetch dashboard overview data
  • bookings.getPending(professionalId, limit) - Get recent pending bookings
  • services.getActive(professionalId) - Fetch active service listings
  • earnings.getMonthly(professionalId) - Calculate monthly earnings
  • reviews.getRecent(professionalId, limit) - Get latest customer reviews
  • analytics.getPerformance(professionalId, period) - Fetch performance metrics
  • notifications.getUnread(professionalId) - Get unread notifications
  • bookings.respondQuick(bookingId, action) - Quick accept/decline from dashboard
