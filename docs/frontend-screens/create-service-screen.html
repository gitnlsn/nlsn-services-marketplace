Create Service Screen

Description

The Create Service Screen enables professionals to list new services on the marketplace. This comprehensive form guides users through all necessary steps to create attractive, complete service listings that effectively communicate their offerings to potential customers and maximize booking opportunities.

Primary Purpose: Provide a user-friendly interface for professionals to create detailed, compelling service listings that attract customers and drive bookings while ensuring all required information is captured.

UI Elements

  • Progress Indicator: Step-by-step progress bar showing completion status
  • Basic Information Section:
    • Service title (required)
    • Category and subcategory dropdowns
    • Service description textarea
    • Keywords/tags input
  • Pricing Section:
    • Pricing model selection (fixed, hourly, custom)
    • Base price input
    • Additional options/add-ons
    • Discount settings
  • Service Details:
    • Duration estimate
    • Service location options
    • What's included list
    • Requirements from customer
    • Cancellation policy selection
  • Media Upload:
    • Primary service image upload
    • Additional image gallery (up to 10)
    • Video upload option
    • Image editing tools
  • Availability Settings:
    • Working hours configuration
    • Blackout dates
    • Advance booking requirements
    • Buffer time between bookings
  • Preview Section: Live preview of how the service will appear to customers
  • Action Buttons: Save as Draft, Preview, Publish Service

API Calls

  • categories.getAll() - Fetch available service categories
  • services.create(serviceData) - Create new service listing
  • services.saveDraft(serviceData) - Save incomplete service as draft
  • media.upload(files) - Upload service images/videos
  • services.validateData(serviceData) - Validate service information
  • pricing.calculateFees(price) - Calculate platform fees
  • location.validateCoverage(area) - Check service area coverage
  • services.checkDuplicates(title, description) - Prevent duplicate listings

Form Validation Rules

  • Required Fields: Title, category, description, price, duration
  • Text Limits: Title (100 chars), description (500 words), requirements (200 words)
  • Price Validation: Minimum price thresholds, maximum pricing limits
  • Image Requirements: Minimum 1 image, max file size 5MB, specific dimensions
  • Duplicate Prevention: Check for similar existing services

Smart Features

  • Auto-Save: Periodic saving of form data to prevent loss
  • Smart Suggestions: AI-powered title and description recommendations
  • Pricing Intelligence: Market rate suggestions based on category
  • SEO Optimization: Keyword suggestions for better discoverability
  • Template Library: Pre-built service templates for common categories

Quality Assurance

  • Content Moderation: Automated screening for inappropriate content
  • Image Analysis: Quality checking and automatic enhancement suggestions
  • Completeness Score: Real-time scoring of listing completeness
  • Review Process: Admin review for first-time service creators
